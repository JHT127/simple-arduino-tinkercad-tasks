# Simple Arduino Tinkercad Tasks

Embedded systems tasks developed as part of **Interfacing Techniques (ENCS4380)** course at **Birzeit University**.

**Student:** Fourth Year Computer Engineering  
**University:** Birzeit University, Palestine  
**Course:** Interfacing Techniques (ENCS4380)

## 📂 Course Projects

### Arduino Fundamentals
1. **Battery Status Indicator** - Register-level I/O control
2. **Timing Comparison** - millis() vs Timer1 interrupts  
3. **Rotary Encoder** - Hardware interrupts for direction detection
4. **Custom LCD Driver** - 4-bit communication without libraries

### Register Level Programming (ATmega328P)
1. **Hardware Interrupts** - External & pin change interrupts
2. **Timer1 CTC Mode** - Precise 1Hz timing
3. **Timer2 Overflow** - Software timing with interrupts
4. **Fast PWM** - LED brightness control
5. **ADC Register** - Potentiometer reading for dimmer

## 🎯 Course Objectives
- Master register-level microcontroller programming
- Implement simulation-to-hardware workflows
- Develop embedded systems with Tinkercad and Arduino
- Understand timing, interrupts, and peripheral communication
